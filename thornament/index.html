Presentation projet "Thornament"

Thornament est un projet de fin de licence en cours de developpement par 5 étudiant en L3 Info. C'est une application web qui une fois terminé permettra à un utilisateur de créer et rejoindre des tournois personnalisés sur les jeux vidéos.

Cette app web est developpée avec ReactJS pour le front-end et Firebase pour le back-end. Le répositoire gitlab est en privé.
